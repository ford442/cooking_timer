# Cooking Timer App - User Guide

## Quick Start

### Creating Your First Timer

1. **Launch the app** - You'll see an empty screen with a + button
2. **Tap the + button** to create a new timer
3. **Enter timer details**:
   - Give it a name (e.g., "Boil Pasta")
   - Set the duration using the time pickers
   - Optionally add notes (e.g., "Stir occasionally")
   - Optionally add a photo (camera or gallery)
4. **Tap Save** to create the timer

### Running a Timer

1. **Tap on a timer** from the main list
2. **Tap Start** to begin countdown
3. Watch the countdown in digital or analog mode
4. Use **Pause** to temporarily stop
5. Use **Reset** to return to original duration
6. Use **Toggle Display** to switch between digital/analog

### Managing Multiple Timers

- Create multiple timers for different cooking steps
- Each timer runs independently
- View all timers on the main screen
- See real-time countdown for all active timers
- Long-press any timer to delete it

## Tips & Tricks

### Creating Effective Timers

1. **Use Clear Names**: "Boil Pasta" instead of "Timer 1"
2. **Add Photos**: Visual reference helps identify timers quickly
3. **Include Notes**: Add reminders like "Stir every 5 min"
4. **Set Exact Times**: Use seconds for precise cooking

### Managing Multiple Timers

1. **Stagger Start Times**: Don't start all timers at once
2. **Check Main Screen**: See all timer states at a glance
3. **Use Pause**: Pause timers when adjusting other tasks
4. **Color Coding**: Running (green), Paused (orange), Done (red)

### Display Mode Selection

- **Digital**: Better for precise time reading
- **Analog**: Better for quick glance at progress
- **Switch Anytime**: Change mode without affecting countdown

## Common Scenarios

### Cooking a Complete Meal

Example: Pasta with Sauce

1. Create "Boil Water" timer (10 minutes)
2. Create "Cook Pasta" timer (12 minutes)
3. Create "Simmer Sauce" timer (20 minutes)
4. Start "Boil Water" first
5. When water boils, start "Cook Pasta" and "Simmer Sauce"
6. All timers count down independently

### Using Photos Effectively

- **Before**: Take photo of raw ingredients
- **Reference**: Take photo of cookbook recipe
- **Result**: Take photo of expected outcome
- **Packaging**: Photo of package for temperature/time reference

### Notes Best Practices

Good notes examples:
- "Preheat oven to 350°F"
- "Flip halfway through"
- "Add cheese in last 5 minutes"
- "Let rest 10 min before serving"

## Troubleshooting

### Timer Doesn't Start
- Check that duration is set (not 00:00:00)
- Try tapping Reset then Start
- Close and reopen timer display

### Photo Not Appearing
- Check camera/storage permissions in device settings
- Ensure photo was successfully taken/selected
- Try taking/selecting photo again

### Timer Disappears
- Timers are lost when app closes (current limitation)
- Don't force-close the app while timers are running
- Keep app in background, not closed

### Multiple Timers Confusing
- Use clear, descriptive names
- Add photos to each timer
- Check status colors (green/orange/red)
- Delete completed timers when done

## Permissions Explained

### Camera Permission
**Why needed:** To take photos of cooking tasks
**When requested:** First time you tap "Take Photo"
**If denied:** You can still create timers, just can't take photos

### Storage Permission
**Why needed:** To select photos from your gallery
**When requested:** First time you tap "Choose Photo"
**If denied:** You can still create timers, just can't upload photos

## Limitations

**Current Version Limitations:**
- Timers are lost when app closes (no persistence)
- No background service (app should stay open)
- No timer groups or categories
- No timer templates or presets
- No sharing or export features

**These may be added in future versions**

## Safety Tips

⚠️ **Important Safety Notes:**

1. **Don't Rely Solely on App**: Use as backup to kitchen timers
2. **Check Regularly**: App notifications may be missed
3. **Keep Device Charged**: Timer stops if device dies
4. **Keep Device Nearby**: Don't leave device in other room
5. **Test First**: Try app with non-critical cooking first

## Getting Help

**App Not Working?**
1. Restart the app
2. Check Android version (needs Android 7.0+)
3. Clear app cache in device settings
4. Reinstall the app
